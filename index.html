About the Project

This tool makes Android mirroring extremely simple. No manual scrcpy setup… no adb errors… everything just works. The auto-installer configures scrcpy, adb, snap, USB permissions, display access and fixes all common errors automatically.

Works on ALL Linux systems and with every Android device from Android 10 to Android 15, including Motorola which fails on older scrcpy.

Installation (Works on ALL Linux)

1. Install packages

sudo apt update sudo apt install -y adb python3-gi gir1.2-gtk-3.0 snapd

2. Install scrcpy

sudo snap install scrcpy sudo snap connect scrcpy:raw-usb sudo snap connect scrcpy:desktop

3. Run auto-installer

chmod +x install.sh ./install.sh

4. Start the app

python3 blackbox_mobile_mirror.py

Fix All Errors

❌ Screen not showing

Run:

xhost +si:localuser:$USER xhost +local:root
❌ Bitrate error
Use: --video-bit-rate 2M
❌ Unauthorized device

Enable USB Debugging and accept RSA prompt.

❌ Wireless connect error
adb tcpip 5555 adb connect PHONE_IP:5555
